Aula de Programação Web

Olá, Django + Tailwind!

Este é um projeto de demonstração simples, rodando em container Docker, com banco SQLite local e estilizado com Tailwind CSS via CDN.

🐍

Django

Framework web em Python, "batteries included" — vem com ORM, admin e tudo mais.

🎨

Tailwind CSS

Utility-first CSS — você compõe estilos direto no HTML, sem inventar nomes de classe.

🐳

Docker Compose

Um único docker compose up e o ambiente está pronto.
